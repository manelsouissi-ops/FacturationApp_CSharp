# FacturationApp

Simple Blazor Server invoice management app.

## Run locally

Requirements:
- .NET 8 SDK
- Visual Studio 2022/2026 or Visual Studio Code

From project root:

```powershell
# restore, build and run
dotnet restore
dotnet build
dotnet run
```

Open the browser at the URL printed by `dotnet run` or run from Visual Studio.
